## рҹ“– About

**RailLive** is a Flutter mobile app built for Indian Railways travelers. Search any train by number or name, follow its live journey station-by-station, check your PNR booking status, and join a real-time chat room with other passengers on the same train вҖ” all in one place.

> рҹҺҜ **Mission** вҖ” Make railway travel smarter, simpler, and more connected.

<br/>

## вңЁ Features

<table>
<tr>
<td width="50%" valign="top">

### рҹ”Қ Train Search
Search by **5-digit train number** or **train name** with instant autocomplete powered by a local `trains.json` database. Recent searches are saved for quick access.

### рҹ“Ў Live Status
Real-time station-by-station tracking with **delay info**, **ETA**, and **auto-refresh every 60 seconds** via the IRCTC RapidAPI.

### рҹҺ« PNR Status
Look up booking status instantly. View coach, berth, and charting details. **Last 10 searches** are saved locally.

</td>
<td width="50%" valign="top">

### рҹ’¬ Train Chat
Per-train chat rooms powered by **Cloud Firestore** вҖ” connect with fellow passengers directly from the train detail screen.

### рҹ‘Ө Profile
Firebase-authenticated user profiles stored in Firestore with photo, name, and email вҖ” fully editable.

### рҹ”” Notifications
Push notifications via **Firebase Cloud Messaging** with local notification support on Android.

</td>

## вҡҷпёҸ Setup

### 1пёҸвғЈ Prerequisites

| Requirement | Details |
|:---|:---|
| **Flutter SDK** | [Install Flutter](https://docs.flutter.dev/get-started/install) вҖ” Dart `^3.12.1` |
| **Firebase Project** | Auth (Email + Phone) В· Firestore В· Cloud Messaging |
| **RapidAPI Keys** | [IRCTC Train API](https://rapidapi.com/indian-railway-irctc/api/indian-railway-irctc) В· [PNR Status API](https://rapidapi.com/indian-railway-irctc/api/irctc-indian-railway-pnr-status) |

<br/>

### 2пёҸвғЈ Environment Variables

Create a `.env` file in the project root *(gitignored вҖ” never commit real keys)*:

```env
RAPID_API_KEY=your_rapidapi_key
RAPID_API_HOST=indian-railway-irctc.p.rapidapi.com
RAPID_API=your_rapidapi_subscription_header_value
RAPID_PNP_API=your_pnr_api_key
```

These map to `lib/config/env.dart`.

<br/>

### 3пёҸвғЈ Firebase Configuration

```bash
dart pub global activate flutterfire_cli
flutterfire configure
```

- Place `google-services.json` in `android/app/`
- `lib/firebase_options.dart` is auto-generated by FlutterFire CLI

<br/>

### 4пёҸвғЈ Run

```bash
flutter run
```

> By default, the app launches the **onboarding flow** (`Screen1`). To skip straight to login, set `MaterialApp` home to `AuthScreen()` in `lib/main.dart`.

## рҹҢҗ APIs

| Endpoint | Host | Purpose |
|:---|:---|:---|
| `GET /api/trains-search/v1/train/{trainNo}` | `RAPID_API_HOST` | Train search & details |
| `GET /api/trains/v1/train/status` | `indian-railway-irctc.p.rapidapi.com` | Live running status |
| `GET /getPNRStatus/{pnr}` | `irctc-indian-railway-pnr-status.p.rapidapi.com` | PNR booking status |

All requests require `x-rapidapi-key` and `x-rapidapi-host` headers.
